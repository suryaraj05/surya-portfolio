# SuryaOS Web Foundation

Frontend foundation for Next.js 15, designed as an AI infrastructure company website system.

## Principles
- Server Components first
- SEO first
- Mobile first
- Accessibility first

## Setup
1. Install dependencies:
   - `npm install`
2. Sync backend contracts:
   - `npm run sync:contracts`
3. Run:
   - `npm run dev`

## Included Foundations
- Design system tokens and theme variables
- Typography system and editorial scale
- Layout primitives (`Container`, `Section`, `Stack`, `SiteShell`)
- Component library baseline (`Button`, `Card`, `Badge`, typography primitives)
- Typed API client layer
- CMS consumption layer
- SEO metadata + sitemap + robots
- Route architecture map
- Framer motion presets
- TanStack Query provider

## Not Included Yet
- Final marketing/product pages
- Final copywriting and imagery
- Detailed case study page implementations
